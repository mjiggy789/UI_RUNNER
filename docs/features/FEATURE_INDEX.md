# Feature Index

This directory contains one document per major runtime/service feature.

1. `01-loader-bootstrap.md`: Script-tag parsing, signed-config fetch, runtime bootstrap.
2. `02-signed-config-and-domain-controls.md`: Signature verification, allowlist gates, kill switch, validity window.
3. `03-runtime-mount-and-overlay.md`: Canvas host mounting, Shadow DOM/iframe fallback, runtime control overlays.
4. `04-dom-sampling-and-colliders.md`: DOM extraction into colliders with one-way/solid heuristics.
5. `05-world-index-and-live-sync.md`: Spatial index, scroll-shift updates, observer-driven rescans.
6. `06-physics-and-movement-controller.md`: Physics integration, collision solving, slide/climb/tuck movement states.
7. `07-ai-navigation-brain.md`: Hierarchical planner + reactive mover, subtargets, recovery, probe-driven decisions.
8. `08-manual-target-mode.md`: User-directed targets with graph routing and scroll-stable page-coordinate anchors.
9. `09-strict-mode.md`: Stronger target commitment and retry behavior.
10. `10-settings-and-keyboard-controls.md`: Runtime hotkeys plus behavior/debug visualization toggles.
11. `11-real-website-mode.md`: MySpace-style long-scroll demo DOM injection mode.
12. `12-debug-renderer-and-hud.md`: Toggleable overlays for calculations, probes, routing, and timers.
13. `14-pause-resume-and-lifecycle.md`: Pause FAB and lifecycle controls.
14. `15-public-api-and-events.md`: `window.ParkourBot` API and emitted runtime events.
15. `16-telemetry.md`: Optional telemetry emission and ingestion.
16. `17-themes-and-sprites.md`: Visual themes and sprite behavior.
17. `18-decision-reference-with-examples.md`: Full decision catalog (triggers, actions, examples, debug/log evidence).
18. `19-stalls-and-loops.md`: Description of stalls and ping-pong loops.
